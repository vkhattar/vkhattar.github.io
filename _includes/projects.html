  Floatout

Crowdsourced photo sharing iOS Application written in Swift with Firebase being the backend. It features asynchronous image caching and loading of images which are hosted on google cloud.

  Buggui!

Created a car that can be directly manipulated by the user using scene graph and affine transformations. The user can rotate, make the axles longer and turn the tires, it uses an HTML 5 canvas and is written in Javascript, Check it out.

  Trancepixel

It is a smart T-shirt that displays light patterns based on the body movements of the person. It is an ideal accessory for concert goers who love technology, art and design. It is made using Arduino based lilypad and can be controlled using an Android application using Bluetooth as the mode of communication.
